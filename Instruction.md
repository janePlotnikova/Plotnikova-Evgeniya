Чтобы добавить ненумерованные списки, необходимые пункты выделить *. Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:

1. Элемент
2. Элемент
